# Documentation Index

Project documentation source of truth for DoubleTime (Screen Time Gambling).

## Core Docs

- **spec.md** — Product rules, tech stack, core logic, wagering model, and locking policy.
- **architecture.md** — Layer boundaries, state ownership, folder structure, naming conventions, and service responsibilities.
- **data_flow.md** — Event-driven sequences for launch, refresh, lock/unlock, game resolve, and day reset.
- **styling.md** — Brand palette, typography tokens, spacing, component styling, motion, copy tone, and accessibility rules.
- **TODO.md** — Phased implementation tracker with acceptance criteria and touch points.

## Game Docs

- **games/blackjack.md** — Blackjack rules, card values, player actions, dealer rules, payout model, and worked examples.
- **games/crash.md** — Crash game rules, multiplier curve, cashout mechanics, and house edge.
- **games/plinko.md** — Plinko rules, risk levels, multiplier tables, and binomial distribution.
- **games/mines.md** — Mines rules, 5x5 grid, fixed multiplier table, and UX flow.
- **games/slots.md** — Slots rules, symbol payouts, weighted outcome table, and RTP.

- **llms/markdown-styling.md** — Markdown formatting constraints for all docs updates.
